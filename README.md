# dotfiles
macOSとUbuntu 20.04(WSL2)用のdotfilesです<br>
他のLinux OSはサポートしていません<br>

## インストールコマンド
```script
bash -c "$(curl -fsSL https://raw.githubusercontent.com/AyamotoKohei/dotfiles/main/install.sh)"
```

## neovimの設定をシンボリック化
```script
ln -s ~/dotfiles/nvim ~/.config
```
